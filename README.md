## ðŸŽ¯ The Goal

Countless talented athletes in India go undiscovered due to geographical and financial barriers. **Project Ekalavya** aims to shatter these barriers by turning any smartphone into a fair, accurate, and accessible sports assessment tool, ensuring that talent, not privilege, determines an athlete's future.
